Usage Guide

Master File Encryptor is an easy to use encryption tool that gets the job done with just 2 clicks.

File Encryption

To encrypt a file, select the file and provide an encryption key to secure access to the file. The encryption key is very important, so we advice you keep it save for decryption purpose. Next, click the "Encrypt file" button and a download button will be generated for you to download the encrypted file. Again, we do not save any encryption key in our system; this is to say that you are in full control of the content of your file.

File Decryption

To decrypt a file, simply select the encrypted file and provide the encryption key with which the file was encrypted. It doesn't matter if the name of the file had been changed; so far the right key was provided, your file will be decrypted and a download button will be generated.
